# 01 相对论运动学与观测时间

本章建立 GRB 余辉推导中最常用的一组运动学关系：bulk Lorentz factor、Doppler factor、relativistic beaming 和 observer-frame arrival time。后续 fireball 加速、external shock deceleration、jet break、structured jet viewing angle 都会反复使用这些关系。

## 1. 速度、Lorentz factor 与 ultrarelativistic 近似

定义 outflow 的 bulk speed 为

```tex
\beta \equiv \frac{v}{c},
```

bulk Lorentz factor 为

```tex
\Gamma \equiv \frac{1}{\sqrt{1-\beta^2}}.
```

当 `Γ ≫ 1` 时，速度可展开为

```tex
\beta = \sqrt{1-\Gamma^{-2}}
      \simeq 1 - \frac{1}{2\Gamma^2}.
```

这个近似是 GRB afterglow 中 `R / 2cΓ²` 时间尺度的来源之一：虽然 ejecta 在 lab frame 中运动时间约为 `R/c`，但 photon 和 shell 都以接近光速向观测者运动，observer-frame 的到达时间差被压缩到 `Γ^{-2}` 量级。

## 2. Doppler factor 与频率 / 时间变换

设 emitting material 相对观测者视线夹角为 `θ`，Doppler factor 定义为

```tex
\delta \equiv \frac{1}{\Gamma(1-\beta\cos\theta)}.
```

对 `Γ ≫ 1` 且 `θ ≪ 1`，使用 `\cos\theta \simeq 1-\theta^2/2` 和 `\beta \simeq 1-1/(2\Gamma^2)`，得到常用小角近似：

```tex
\delta \simeq \frac{2\Gamma}{1+\Gamma^2\theta^2}.
```

因此在 `θ = 0` 的 on-axis 情况下 `δ ≃ 2Γ`；当 `θ ≳ Γ^{-1}` 时，Doppler boosting 迅速下降。这就是 relativistic beaming 的核心图像：一个 Lorentz factor 为 `Γ` 的发射区，主要被观测到的角尺度约为

```tex
\theta_{\rm beam} \sim \Gamma^{-1}.
```

若 comoving frame 中 photon frequency 为 `ν'`、时间间隔为 `dt'`，则 observer frame 中近似有

```tex
\nu_{\rm obs} = \frac{\delta}{1+z}\nu',
```

```tex
dt_{\rm obs} = \frac{1+z}{\delta}dt'.
```

这里 `(1+z)` 是 cosmological redshift factor；`δ` 描述局域相对论 Doppler boost。后续比较不同 source 的 `E_iso`、peak frequency、variability time 时，必须区分这两个因子。

## 3. Arrival time relation

考虑 lab frame 中半径 `R` 处、夹角 `θ` 的 surface element 向远处观测者发射 photon。若 central engine 在 `t=0`、`R=0` 发出参考 photon，则该 photon 的 observer-frame 到达时间可写成

```tex
t_{\rm obs} = (1+z)\left(t_{\rm lab} - \frac{R\cos\theta}{c}\right).
```

对以速度 `βc` 运动的 shell，`R \simeq \beta c t_{\rm lab}`，因此

```tex
t_{\rm obs} \simeq (1+z)\frac{R}{c}\left(\frac{1}{\beta}-\cos\theta\right).
```

在 ultrarelativistic 和 small-angle 极限下，

```tex
t_{\rm obs} \simeq (1+z)\frac{R}{2c\Gamma^2}\left(1+\Gamma^2\theta^2\right).
```

on-axis 时得到最常见的量级关系：

```tex
t_{\rm obs,on-axis} \sim (1+z)\frac{R}{2c\Gamma^2}.
```

不同文献中也常见 `R / 4cΓ²` 或带数值系数的版本；差别通常来自是否把 `Γ` 指为 shocked fluid Lorentz factor、shock front Lorentz factor，或是否已经代入 Blandford-McKee self-similar dynamics。使用公式时应记录 Lorentz factor 的定义。

## 4. Equal-arrival-time surface

固定同一个 `t_obs` 时，来自更大 `θ` 的 photon 可以与较小半径或更早 lab-frame 时间的 photon 同时到达观测者。这形成 equal-arrival-time surface。余辉光变曲线不是单一半径瞬时发射的直接投影，而是不同角度、半径、Doppler factor 和局域 emissivity 的积分结果。

这点在三个地方特别重要：

1. **early afterglow onset**：deceleration radius 与 `t_dec` 的转换依赖 `R / Γ² c` 量级关系。
2. **jet break**：当 `Γ` 降到 `Γ \sim θ_j^{-1}`，观测者开始看到 jet edge 或 angular structure 的变化。
3. **structured jet / off-axis viewing**：`θ_v` 改变 Doppler factor 和 arrival-time distribution，使同一 angular energy profile 在不同 viewing angle 下呈现不同 `E_iso`、peak time 和 decay slope。

## 5. 常见误区

- **把 `Γ` 和 `δ` 混用**：`Γ` 是流体 bulk Lorentz factor；`δ` 还依赖 viewing angle。
- **忘记 redshift factor**：局域 Doppler boost 与 cosmological time dilation 同时影响 observed time / frequency。
- **把 `R/c` 当成 observed time**：relativistic shell 的 observed timescale 通常被压缩到 `R / 2cΓ²` 量级。
- **忽略数值系数的定义差异**：`2`、`4`、`8` 等系数常来自 shell / shock / shocked-fluid Lorentz factor 的不同约定，不能脱离 source notation 直接比较。
- **把 beaming cone 当成真实 jet opening angle**：`Γ^{-1}` 是相对论可见角尺度；`θ_j` 是几何开角，二者在 jet break 条件中才被联系起来。

## 6. 和后续章节的接口

- Fireball acceleration：需要用 `Γ` 描述 baryon-loaded outflow 从 acceleration 到 coasting 的演化。
- External shock：`t_obs ∼ R / 2cΓ²` 把 deceleration radius 转换为 deceleration time。
- Synchrotron afterglow：`ν_obs = δν'/(1+z)` 连接 comoving synchrotron frequency 与观测频率。
- Jet break：`Γ(t_j) θ_j ∼ 1` 是准直修正和 true energy estimate 的入口。
- Two-component / structured jet：不同 angular component 的 `Γ(θ)` 和 `dE/dΩ(θ)` 通过 `δ(θ, θ_v)` 投影到 prompt / afterglow observables。
