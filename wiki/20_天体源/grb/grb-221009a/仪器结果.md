# GRB 221009A 仪器结果

## Fermi-GBM

Lesage et al. 的 Fermi-GBM 分析给出的仪器相关结果：

- GBM trigger time 为 2022-10-09 13:16:59.99 UTC。
- 作者报告 trigger time 前未探测到 emission。
- prompt emission 持续超过 600 s。
- afterglow 在 GBM 的 8 keV–40 MeV 能段可见。
- trigger pulse 有显著 emission 到约 15 MeV。
- 采用 z = 0.151 时，Eγ,iso = 1.0×10^55 erg，Lγ,iso = 9.9×10^53 erg/s。

## Konus-Wind

Frederiks et al. 的 Konus-Wind 分析给出的仪器相关结果：

- Konus-Wind trigger time 为 2022-10-09 13:17:01.648 UTC；Earth-crossing reference time 为 13:17:00.401 UTC。
- 论文将 GRB 221009A 描述为 Konus-Wind 超过 28 年观测中探测到的最亮 gamma-ray burst。
- pulsed prompt phase 持续约 600 s，之后的 steady power-law decay 持续超过 25 ks。
- time-averaged Ep ≈ 2.6 MeV；最亮 emission peak 的 Ep ≈ 3.0 MeV。
- 20 keV–10 MeV total energy fluence ≈ 0.22 erg cm^-2；同能段 peak energy flux ≈ 0.031 erg cm^-2 s^-1。
- 采用 z = 0.151 时，Eiso ≈ 1.2×10^55 erg，64 ms 尺度 Liso ≈ 3.4×10^54 erg s^-1。

## SRG/ART-XC

Frederiks et al. 的 ART-XC 相关结果：

- ART-XC 在 2022-10-09 13:19:55 UTC 探测到 GRB 221009A。
- 事件发生在 ART-XC 视场外，信号穿过望远镜侧向结构后被七个探测器记录。
- 作者将 ART-XC light curve 与 Konus-Wind light curve 联合使用，以恢复 prompt emission 的时间结构。

## INTEGRAL/IBIS-PICsIT

Rodi & Ubertini 的 PICsIT 相关结果：

- 该 source 使用 PICsIT 200–2600 keV spectral-timing data 研究 prompt phase 和 early afterglow period。
- PICsIT spectral-timing data 提供 8 个宽能带和 7.8 ms 时间分辨率。
- GRB 221009A 方向约偏离 INTEGRAL pointing direction 65.8°。
- prompt emission 持续超过 600 s，包括 precursor。
- afterglow emission 在约 T0 + 630 s 开始占主导，decay slope 为 1.6 ± 0.2。
- 作者报告无 time bins 接近 telemetry maximum，因此 PICsIT 未受 pile-up effects 影响；但 bright prompt pulses 存在 telemetry gaps。

## LHAASO

LHAASO Collaboration 的 GRB 221009A TeV afterglow 论文给出的仪器相关结果：

- 事件发生在 LHAASO 视场内。
- LHAASO 在最初 3000 s 内探测到超过 64,000 个能量高于 0.2 TeV 的光子。
- 论文基于该 TeV 样本给出光变行为：trigger 后数分钟开始、开始后约 10 s 达峰、峰后衰减、约峰后 650 s 衰减变快。

## Caveat

- 本页记录仪器观测结果和对应时间/能段行为。
- shock-breakout、external shock onset、normal-long-GRB interpretation、flux-tracking physical interpretation、narrow jet / structured jet core 都属于相应论文的解释或表征，应见 [模型解释](模型解释.md) 或 [瞬时辐射](瞬时辐射.md) 的解释边界说明。
- ART-XC 对 GRB 221009A 是 off-axis / through-structure detection，不应写成常规视场内成像观测。
