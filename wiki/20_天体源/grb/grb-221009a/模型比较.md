# GRB 221009A 模型比较

## 本页定位

本页在事件内部比较不同论文对 GRB 221009A afterglow 的模型解释。通用理论推导不在本页重复，优先链接到 `wiki_textbook/grb-afterglow/`；本页只记录 source-specific assumptions、数据约束、拟合参数性质、模型改动和失败点。

## 第一批可复现问题

### 1. Broken power-law light curve toy fit

目标：先不解释物理，只复现 TeV / X-ray / soft gamma-ray 衰减斜率与 break 的 phenomenological fitting。

需要数据：LHAASO TeV light curve、PICsIT early afterglow decay、O'Connor X-ray decay points 或 slope summary。

对应代码：`reproduce/grb/core/fitting/lightcurve.py`。

### 2. Standard forward-shock scaling check

目标：用 `E_k`、`n/A_*`、`epsilon_e`、`epsilon_B`、`p` 的 analytic scalings 生成最小 broadband trend，检查 radio / X-ray / TeV 是否能由单一 forward shock 粗略解释。

对应理论：`04-external-shock动力学.md`、`05-synchrotron余辉谱.md`、`06-light-curve-closure-relations.md`、`17-afterglow-fitting-workflow与参数简并.md`。

对应代码：`reproduce/grb/models/forward_shock/analytic_scalings.py`。

### 3. Extra radio component diagnosis

目标：先复现 Laskar et al. 的“单一 forward shock 不足”这一诊断，而不是立即拟合完整物理 component。

需要：radio SED / light curve 数据文件。

### 4. Structured jet comparison scaffold

目标：先记录 O'Connor shallow structured jet 与 LHAASO narrow core 的参数定义差异，暂不实现完整 structured jet 数值积分。

后续代码：`reproduce/grb/models/structured_jet/`。
