# IACT 分析流程

## 当前流程骨架

de Naurois 2006 把 Atmospheric Cherenkov Telescope analysis 分成三类代表性方法：Hillas-parameter based analysis、Model analysis 和 3D Model analysis。这三类方法使用的信息不同，因此可比较、也可组合。

## Hillas / scaled-cuts 路线

1. 从 camera image 中提取二维椭圆参数：length、width、size、nominal distance、main-axis azimuth、orientation angle。
2. 单望远镜情况下用 length、size 等参数估计 shower direction，但 major axis 两侧存在 head-tail degeneracy。
3. stereo 情况下用多个 telescope image axes 的交点估计 source direction，并几何重建 shower impact。
4. gamma/hadron separation 使用 scaled cuts：把 observed width / length 与 Monte Carlo expectation 比较，得到 Scaled Width、Scaled Length，再组合为 Mean Scaled Width / Mean Scaled Length。

## Model analysis 路线

Model analysis 使用 semi-analytical shower development model 生成 template，并逐 pixel 比较观测 image 与模型 image。该方法通过 maximum likelihood 使用所有 camera pixels，可在不要求 image cleaning 的情况下重建 primary energy、direction 和 impact。

de Naurois & Rolland 2009 把这一路线细化为 H.E.S.S. likelihood reconstruction：先由 shower image model 给出每个 pixel 的 expected signal `μ`，再用包含 Poisson photoelectron statistics、pedestal width `σ_p` 和 single-photoelectron resolution `σ_γ` 的 pixel likelihood 拟合 direction、impact、first interaction depth 和 energy。其背景抑制量包括 goodness-of-fit `G`、ShowerGoodness、BackgroundGoodness 和 reconstructed primary interaction depth。

de Naurois 2006 已指出 Model analysis 显式处理 night sky background 和 electronics pedestal，因此相对于 Hillas / 3D Model analysis，在高 NSB 下 efficiency 下降更平缓；de Naurois & Rolland 2009 进一步把该稳定性归因到逐 pixel 使用 measured `σ_p` 和 `σ_γ`，并报告 ShowerGoodness 在约 300 MHz NSB 内无强演化。

## 3D Model analysis 路线

3D Model analysis 将 shower 描述为三维 Gaussian photosphere，并用 line-of-sight integral 预测每个 pixel 的 collected light。它重建 mean altitude、impact、direction、3D width / length 和 luminosity。

该方法在 fit 过程中利用 rotational symmetry 可拒绝约 70% hadrons；剩余事件中 shower width 是重要的 gamma/hadron discriminator。de Naurois 2006 将 3D Model analysis 视为 Hillas parameters 的三维推广。

## 方法比较 caveat

- Model 和 3D Model analyses 在 de Naurois 2006 的比较中有约 20% 更高 gamma-ray efficiency，但也保留更多 background，因此单独 sensitivity 接近。
- Hillas analysis 对 off-axis truncated images 更敏感；Model 和 3D Model analysis 由于可 extrapolate available information，off-axis efficiency 更平坦。
- Hillas 和 3D Model efficiency 在 NSB 超过约 200 MHz 后下降较快；Model analysis 对 NSB 更稳健。
- de Naurois & Rolland 2009 报告 H.E.S.S. Model Analysis 相对 standard Hillas reconstruction 的 sensitivity 接近提升 factor 2，但这是 H.E.S.S. data、cuts 和 model implementation 下的结果。
- 不同方法对 shower 的不同信息敏感；早期 Model Analysis 可通过与 Mean Scaled Width / Length 组合提升 rejection，但 2009 paper 指出改进后的 ShowerGoodness cut 已吸收大部分额外判别能力。
