# GRB 080319B 模型解释

## Narrow core + wider jet 图像

Racusin et al. 2008 将 GRB 080319B 解释为观测者接近 very narrow core 轴向的事件。source interpretation 中的代表性参数为：

| 成分 | opening angle | 主要作用 |
| --- | --- | --- |
| very narrow core | 约 `0.2°` | 解释 naked-eye prompt optical brightness 和极端 apparent brightness。 |
| wider component | 约 `4°` | 解释后续 afterglow 行为。 |

该图像可以看作 two-component / structured outflow 的事件级案例：narrow core 产生极端 on-axis brightness，wider component 控制后续较宽角度的 afterglow evolution。

## Prompt optical / gamma mismatch 的含义

Prompt optical flux 远高于 gamma-ray spectrum 的低能外推，说明 prompt optical 和 gamma-ray emission 至少需要 distinct spectral components。该结论支持多成分辐射图像，但它本身不唯一决定 narrow core / wider jet geometry；geometry 需要结合 afterglow modeling 才能约束。

## Caveats

- narrow core / wider component 是模型解释，不是直接成像。
- opening angles、Lorentz factors 和 wind-like medium 依赖 microphysics、density profile 和 afterglow fitting assumptions。
- optical / gamma spectral mismatch 排除了简单单成分谱外推，但并不唯一指定辐射机制或 jet angular profile。
