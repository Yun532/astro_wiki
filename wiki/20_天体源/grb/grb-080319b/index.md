# GRB 080319B

## 页面定位

GRB 080319B 是 Racusin et al. 2008 研究的 “naked-eye” gamma-ray burst。该事件的 prompt optical emission 达到约 visual magnitude `5.3`，红移约 `z ≈ 0.937`。它在本知识库中的主要用途是作为 narrow core + wider jet / two-component outflow 的事件级案例：极亮 prompt optical、optical / gamma-ray spectral mismatch 和 broadband afterglow 被放入双成分或结构化喷流框架中解释。

## 使用 caveat

GRB 080319B 的 optical / gamma-ray mismatch 是观测和谱能分布约束；very narrow core、wider component、opening angle 和 wind-like environment 是 Racusin et al. 的模型解释。该事件支持多 spectral components 和 structured / two-component outflow 的必要性，但不唯一决定辐射机制或角向结构。
