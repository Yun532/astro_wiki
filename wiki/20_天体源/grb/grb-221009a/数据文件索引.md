# GRB 221009A 数据文件索引

## 本页定位

本页只登记可取用或待取用的数据资产，不把模型解释写成观测事实。每条记录都应尽量给出来源、路径、数据类型、单位、时间零点、能段、处理状态和 caveat。

处理状态约定：

| 状态 | 含义 |
| --- | --- |
| raw | 原始 PDF、source 包、supplement、公开数据入口，尚未抽表或清洗。 |
| extracted | 已从 source / table / figure 提取，但尚未统一单位或时间零点。 |
| model-ready | 已整理成可直接用于拟合或作图的表格。 |
| digitized | 从图像数字化得到，必须保留 digitization 方法和误差 caveat。 |
| pending | 已确认存在数据线索，但尚未提取。 |

## Figure-derived data policy

若 source 包或 supplement 没有公开数值表，允许 digitize figure，但必须单独记录：原 figure 路径、digitization 工具、坐标轴单位和缩放、估计误差、是否只用于 toy reproduction。Digitized 点不得当作论文官方数据表。

## 与复现代码的关系

## 下一步提取优先级

1. LHAASO TeV light curve：先用于 broken power-law / onset-break toy fit。
2. Laskar radio-to-GeV multiwavelength light curve：用于 forward-shock first-pass closure / SED 检查；Liverpool Telescope、Swift/UVOT 和 Fermi/LAT 表已先抽取为 `laskar_lt_photometry.csv`、`laskar_uvot_photometry.csv`、`laskar_lat_flux.csv`。
3. Konus-Wind prompt spectral fits：`konus_band_spectral_fits.csv` 已抽取，可先用于 prompt Band 参数、峰值能量和 flux 的交叉比较；完整 energetics 仍需单独定义积分区间和宇宙学参数。
4. O'Connor X-ray decay：用于 shallow structured jet 与 standard afterglow 的比较。
5. INTEGRAL/PICsIT early afterglow：用于 prompt-to-afterglow 分界和 soft gamma-ray decay slope 对照。
