# GRB 221009A 瞬时辐射

## Fermi-GBM prompt emission

Lesage et al. 报告 Fermi-GBM 对 GRB 221009A 的发现和分析，并将该事件描述为 GBM 观测到的最高 flux gamma-ray burst。

关键观测事实：

- GBM trigger time 为 2022-10-09 13:16:59.99 UTC，论文将其作为 prompt emission onset。
- 作者报告 trigger time 之前未探测到 emission。
- prompt emission 持续超过 600 s。
- trigger pulse 有显著 emission 到约 15 MeV。
- 论文报告 prompt phase 在 GBM 能段平滑过渡到 afterglow。

## Konus-Wind / SRG-ART-XC prompt emission

Frederiks et al. 报告 Konus-Wind 与 SRG/ART-XC 对 GRB 221009A 的联合观测。

关键观测事实：

- Konus-Wind trigger time 为 2022-10-09 13:17:01.648 UTC；论文采用的 Earth-crossing reference time 为 13:17:00.401 UTC。
- prompt emission time profile 包含 initial pulse、quiescent interval 和约 450 s 的极亮 emission complex。
- pulsed prompt phase 持续约 600 s。
- prompt phase 后转入 steady power-law decay，持续超过 25 ks。
- ART-XC 在视场外记录到该事件，并用于约束 light curve shape。

## INTEGRAL/IBIS-PICsIT prompt emission

Rodi & Ubertini 使用 INTEGRAL/IBIS-PICsIT 分析 200–2600 keV prompt 和 early afterglow emission。

关键观测事实：

- GRB 221009A 的 precursor 在 T0 = 13:17:00 UTC 附近被 PICsIT 清楚探测到。
- prompt emission 持续超过 600 s，包括 precursor。
- PICsIT 200–1200 keV light curve 显示多个 pulse；部分 bright pulses 受 telemetry gaps 影响。
- 论文报告 spectral evolution 具有 flux-tracking behavior，即亮度较高时 spectrum 较 soft。
- afterglow emission 在约 T0 + 630 s 开始占主导。

## 解释边界

- “thermal/photospheric shock-breakout emission” 是 Lesage et al. 对 trigger pulse 特征的解释，不应写成直接观测事实。
- `t0+600 s` 被 Lesage et al. 表征为 external shock onset；该表征应与观测到的 prompt duration 和 afterglow transition 区分。
- “very hard, super-energetic version of a normal long GRB” 是 Frederiks et al. 对 Amati / Yonetoku 关系一致性的解释，不是独立观测事实。
- Rodi & Ubertini 对不同 pulses 可能对应不同 emission processes 的讨论属于解释；本页只将 flux-tracking 和 changing spectral-index / flux relation 作为谱演化结果记录。
- ART-XC 的 GRB 221009A 观测是 off-axis / through-structure detection，不能写成常规视场内成像观测。
