# GRB 221009A 能谱演化

## Fermi-GBM 能段和高能延伸

Lesage et al. 报告 GRB 221009A 的 afterglow 在 Fermi-GBM 的 8 keV–40 MeV 能段可见，并指出 prompt phase 在该能段平滑过渡到 afterglow。

已整理事实：

- GBM 能段：8 keV–40 MeV。
- trigger pulse 有显著 emission 到约 15 MeV。
- prompt emission 持续超过 600 s，并在 GBM 能段过渡到 afterglow。
- 采用 z = 0.151 时，论文报告 Eγ,iso = 1.0×10^55 erg，Lγ,iso = 9.9×10^53 erg/s。

## Konus-Wind 能谱和 energetics

Frederiks et al. 基于 Konus-Wind spectral data 与 Konus-Wind / ART-XC light curves 给出 GRB 221009A 的 prompt spectral peak、fluence 和 isotropic-equivalent energetics。

已整理事实：

- time-averaged spectral peak energy：Ep ≈ 2.6 MeV。
- 最亮 emission peak 的 Ep ≈ 3.0 MeV。
- 20 keV–10 MeV 总 energy fluence ≈ 0.22 erg cm^-2。
- 同能段 peak energy flux ≈ 0.031 erg cm^-2 s^-1。
- 采用 z = 0.151 时，Eiso ≈ 1.2×10^55 erg，64 ms 尺度 Liso ≈ 3.4×10^54 erg s^-1。

## INTEGRAL/IBIS-PICsIT 软 gamma-ray 谱演化

Rodi & Ubertini 使用 PICsIT 200–2600 keV spectral-timing data 研究 prompt phase 和 early afterglow period。

已整理事实：

- PICsIT 的 spectral-timing data 覆盖 200–2600 keV，论文在 7 个 250–2600 keV channel 上用 0.5 s integration 拟合谱参数。
- 论文报告 flux-tracking behavior：亮度较高时 spectrum 较 soft，photon index 随 flux 增加总体增大。
- spectral-index / flux relation 在 burst 过程中改变；不同 pulse、同一 pulse 的 rise 与 decay phase 也可表现出不同关系。
- afterglow emission 在约 T0 + 630 s 开始占主导，衰减 slope 为 1.6 ± 0.2。

## 解释边界

- Fermi-GBM 的 Eγ,iso / Lγ,iso 与 Konus-Wind 的 Eiso / Liso 来自不同仪器、能段和分析定义，比较时必须保留 source 和能段。
- Frederiks et al. 将 isotropic energetics 与 Amati / Yonetoku 关系相联系，并解释为 very hard、super-energetic version of a normal long GRB；该判断应归因给作者。
- Rodi & Ubertini 对 flux-tracking 的物理解释涉及 synchrotron / photospheric emission 等机制，属于模型解释或讨论，不应写成观测事实。

## 待补充

- 后续 INTEGRAL / Konus-Wind / Fermi-GBM / Fermi-LAT source 全部 ingest 后，应建立跨仪器能段和 spectral-index / Ep 对照表。
