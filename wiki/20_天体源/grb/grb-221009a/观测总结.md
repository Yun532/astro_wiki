# GRB 221009A 观测总结

## 已整理观测事实

### Fermi-GBM prompt / MeV-band afterglow

Lesage et al. 报告 Fermi-GBM 对 GRB 221009A 的发现和分析，并将该事件描述为 GBM 观测到的最高 flux gamma-ray burst。已整理事实：

- GBM trigger time 为 2022-10-09 13:16:59.99 UTC，论文将其作为 prompt emission onset。
- 作者报告 trigger time 之前未探测到 emission。
- prompt emission 持续超过 600 s。
- afterglow 在 GBM 的 8 keV–40 MeV 能段可见。
- trigger pulse 有显著 emission 到约 15 MeV。
- 采用 z = 0.151 时，论文报告 Eγ,iso = 1.0×10^55 erg，Lγ,iso = 9.9×10^53 erg/s。

### Konus-Wind / SRG-ART-XC prompt 和 extended emission

Frederiks et al. 报告 Konus-Wind 与 SRG/ART-XC 对 GRB 221009A 的联合观测，并将其描述为 Konus-Wind 超过 28 年观测中探测到的最亮 gamma-ray burst。已整理事实：

- Konus-Wind trigger time 为 2022-10-09 13:17:01.648 UTC；论文换算的 Earth-crossing reference time 为 13:17:00.401 UTC。
- pulsed prompt phase 持续约 600 s。
- prompt phase 后出现 steady power-law decay，持续超过 25 ks。
- time-averaged spectral peak energy 为 Ep ≈ 2.6 MeV，最亮 emission peak 的 Ep ≈ 3.0 MeV。
- 20 keV–10 MeV 总 fluence ≈ 0.22 erg cm^-2，同能段 peak energy flux ≈ 0.031 erg cm^-2 s^-1。
- 采用 z = 0.151 时，论文给出 Eiso ≈ 1.2×10^55 erg，64 ms 尺度 Liso ≈ 3.4×10^54 erg s^-1。
- ART-XC 在视场外记录到该事件，主要为 light curve shape 提供约束。

### INTEGRAL/IBIS-PICsIT soft gamma-ray 演化

Rodi & Ubertini 使用 INTEGRAL/IBIS-PICsIT spectral-timing data 分析 GRB 221009A 在 200–2600 keV 的 prompt 和 early afterglow 行为。已整理事实：

- GRB 221009A 发生时 INTEGRAL 约偏离该源方向 65.8°。
- PICsIT 数据显示 prompt emission 持续超过 600 s，包括 precursor。
- 论文报告 flux-tracking spectral behavior：亮度越高时 spectrum 越 soft。
- spectral-index / flux relation 在 burst 过程中发生变化，并在不同 pulse 及 rise/decay phase 中不同。
- afterglow emission 在约 T0 + 630 s 开始占主导，并以 1.6 ± 0.2 的 slope 衰减。
- 最亮 prompt pulses 的部分时间段有 telemetry gaps；作者报告 PICsIT 未受 pile-up effects 影响。

### Laskar et al. radio-to-GeV 多波段余辉

Laskar et al. 报告 GRB 221009A 的 radio-to-GeV afterglow observations，覆盖 radio 到 gamma rays，横跨约 15 个数量级的 photon energy。已整理事实：

- 数据集包括 radio/mm、optical/UV、X-ray / hard X-ray，以及 GeV / gamma-ray afterglow 约束。
- radio/mm 数据包括 MeerKAT、VLA、VLBA、ATCA、ALMA、NOEMA、SMA 等观测来源。
- 论文框架中，观测 campaign 覆盖约四个数量级的时间，并延伸到约 100 d。
- radio/mm 数据对 forward-shock afterglow model 提供强约束。
- 作者指出 radio/mm 数据不能由单一 standard forward shock model 完整解释，需要额外 emission component。
- ATCA 数据在该 source 中被作者列为 completeness，但作者提示 calibration 限制，不应在未复核前作为强约束使用。

### LHAASO TeV 余辉观测

LHAASO Collaboration 报告，GRB 221009A 发生在 LHAASO 视场内，因此 LHAASO 得以记录其 TeV afterglow。该论文给出的核心观测事实包括：

- LHAASO 在最初 3000 s 内探测到超过 64,000 个能量高于 0.2 TeV 的光子。
- TeV photon flux 在 GRB trigger 后数分钟开始。
- TeV flux 在开始后约 10 s 达到峰值。
- 峰值后 TeV emission 进入衰减阶段。
- 衰减在约峰后 650 s 变快。

## 证据边界

- Fermi-GBM source 主要支持 trigger、prompt emission、MeV-band afterglow、energetics 和 prompt-to-afterglow transition。
- Konus-Wind / ART-XC source 主要支持 prompt fluence、peak flux、Ep、约 600 s pulsed prompt phase、超过 25 ks 的 extended decay，以及 ART-XC light curve shape 约束。
- INTEGRAL/PICsIT source 主要支持 200–2600 keV spectral-timing evolution、flux-tracking behavior、T0 + 630 s afterglow dominance 和 telemetry / pile-up caveat。
- Laskar et al. source 主要支持 radio-to-GeV 多波段余辉数据范围、radio/mm 约束和额外 radio component 问题；forward-shock、wind-like medium、equipartition mass / Lorentz factor / energy 是模型解释。
- LHAASO source 主要支持 TeV afterglow photon sample、TeV light curve 和 narrow jet 模型解释。
- shock-breakout、external shock onset、normal long GRB / Amati-Yonetoku interpretation、flux-tracking 的物理解释、narrow jet、forward shock / wind-like medium 和 additional component 参数都应作为相应 source 的解释或表征，不作为未归因观测事实使用。
