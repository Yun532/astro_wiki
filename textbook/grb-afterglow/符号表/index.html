GRB 余辉符号表

Section titled “GRB 余辉符号表”
符号含义常用单位 / 归一化注意事项
Γbulk Lorentz factordimensionless注意区分 initial Γ_0、shock Γ、component Γ。
βv/cdimensionlessβ = sqrt(1 - Γ^-2)。
δDoppler factordimensionless依赖 viewing angle 和 Γ；δ = [Γ(1-β cosθ)]^-1。
θgeneric angle / jet polar anglerad简单 beaming 中可表示 emitting element 与视线夹角；structured jet 中表示相对 jet axis 的 polar angle，需按上下文区分。
θ_jjet opening half-anglerad 或 degreesmall-angle approximation 下 f_b ≈ θ_j^2/2。
θ_cstructured jet core anglerad 或 degreecore / wing 分界的模型参数，不等同于 top-hat sharp edge。
θ_nnarrow component opening half-anglerad 或 degreetwo-component jet 中窄而快的 core opening angle。
θ_wwide component opening half-anglerad 或 degreetwo-component jet 中较宽较慢 component 的 opening angle。
θ_vviewing anglerad 或 degree观测者视线与 jet axis 的夹角；也常写作 θ_obs。
ψemitting element 与观测者视线夹角radstructured jet 积分中由 θ、φ、θ_v 决定，进入 Doppler factor。
𝓔(θ)angular energy profileerg sr^-1𝓔(θ)=dE/dΩ；structured jet 的核心函数。
E_isoisotropic-equivalent energyerg观测推断量，不等于 true energy。
Eisotropic-equivalent or beaming-corrected kinetic energyerg必须说明是动力学中使用的等效能量还是真实能量。
E_nnarrow component energyergtwo-component jet 中窄成分能量；需说明 iso 还是 true。
E_wwide component energyergtwo-component jet 中宽成分能量；可比 narrow component true energy 更大。
E_0fireball 初始总能量erg用于定义 dimensionless entropy。
M_0fireball 初始 baryonic massg 或 M_sunbaryon loading 的核心量。
ηdimensionless entropydimensionlessη = E_0 / (M_0 c²)；baryonic fireball 中常给出 terminal Γ 的量级。
Γ_0coasting shell 初始 Lorentz factordimensionlessafterglow onset / deceleration 公式中的初始 Lorentz factor。
Γ_{0,n}narrow component initial Lorentz factordimensionlesstwo-component jet 中窄成分通常更高。
Γ_{0,w}wide component initial Lorentz factordimensionlesstwo-component jet 中宽成分通常较低，使 t_dec,w 较晚。
R_0central-engine 初始半径尺度cmfireball acceleration scaling 的归一化半径。
R_ssaturation radiuscmR_s ~ η R_0，加速转入 coasting 的量级半径。
R_phphotosphere radiuscmThomson optical depth 降到约 1 的半径。
R_decdeceleration radiuscmswept-up mass 足以显著减速 ejecta 的半径。
nISM number densitycm^-3wind medium 中通常改用 A 或 A_*。
n_extshock 前外部介质 number densitycm^-3shock jump condition 中的 upstream density。
n’shocked fluid comoving number densitycm^-3ultra-relativistic strong shock 中量级 n' ~ 4Γ n_ext。
e’shocked fluid comoving internal energy densityerg cm^-3afterglow synchrotron magnetic field 和 electron energy 的入口。
kexternal density power-law indexdimensionlessρ_ext = A R^-k；ISM 为 0，wind 为 2；general-k closure 和 density-diagnosis 中需明确假设。
ρ_extcircumburst medium mass densityg cm^-3可写作 n m_p、A R^-2 或 general A R^-k。
m_swswept-up external massgdeceleration 和 BM energy scaling 的核心量。
ε_eelectron energy fractiondimensionlessshock internal energy 中进入 non-thermal electrons 的比例。
ε_Bmagnetic energy fractiondimensionlessB'^2/(8π)=ε_B e'。
B’comoving magnetic fieldGshocked fluid frame 中的磁场。
pelectron power-law indexdimensionlessN(γ_e) ∝ γ_e^-p；不要和 photon index 混淆。
γ_eelectron Lorentz factordimensionlesscomoving frame electron energy。
γ_mminimum injected electron Lorentz factordimensionless对应 synchrotron characteristic frequency ν_m。
γ_ccooling electron Lorentz factordimensionlessdynamical time 内刚好显著冷却的 electron Lorentz factor。
ν_msynchrotron injection / characteristic frequencyHz由 γ_m 对应的 spectral break。
ν_csynchrotron cooling frequencyHz由 γ_c 对应的 spectral break。
ν_asynchrotron self-absorption frequencyHzradio band 中常重要，本章只作为接口。
F_ν,maxpeak spectral flux densityJy 或 erg cm^-2 s^-1 Hz^-1afterglow synchrotron spectrum 的归一化。
YCompton parameterdimensionlessinverse Compton cooling 对 γ_c 的修正；SSC 章节中需区分 Thomson 与 Klein-Nishina regime。
αtemporal decay indexdimensionless本教材采用 F_ν ∝ t^-α ν^-β_spec。
β_specspectral indexdimensionless为避免和速度 β=v/c 混淆，谱指数写作 β_spec。
Γ_phphoton indexdimensionlessX-ray 文献中常有 β_spec = Γ_ph - 1，需检查定义。
t_jjet-break times, d常由 Γ(t_j) ~ θ_j^-1 估计。
t_{jet,n}narrow component jet-break times, dtwo-component jet 中窄成分 jet break；可能被 wide component 掩盖。
t_{dec,w}wide component deceleration times, dtwo-component jet 中宽成分开始显著贡献 afterglow 的关键时间。
f_bbeaming fractiondimensionless小角近似常用 f_b ≈ θ_j²/2，需检查 one-sided / two-sided convention。
E_γbeaming-corrected prompt gamma-ray energyerg常由 f_b E_{γ,iso} 得到。
E_kafterglow kinetic energyerg需说明是 isotropic-equivalent 还是 beaming-corrected。
η_γprompt radiative efficiencydimensionless常定义为 E_γ/(E_γ+E_k)。
L_injenergy-injection luminosityerg s^-1常用 L_inj ∝ t^-q 参数化。
qinjection luminosity temporal indexdimensionlessq<1 时累计注入能量随时间增长。
eblast-wave energy growth indexdimensionless若 E(t)∝t^e，连续注入中常有 e=1-q。
sejecta Lorentz-factor stratification indexdimensionlessrefreshed shock 中常用 E(>Γ)∝Γ^-s。
T_sdspin-down timescalesmagnetar-like injection 的 characteristic timescale。
t_binjection-break / plateau-end times, denergy injection 结束时的光变转折时间。
t_\timesreverse-shock crossing times, dreverse shock 穿过 ejecta 的时间；常与 max(T,t_dec) 比较。
σmagnetization parameterdimensionlessPoynting flux 与 matter energy flux 的比值；高 σ 可改变或抑制 reverse shock。
ν_{m,r}reverse-shock characteristic frequencyHzreverse shock 的 injection break；常与 forward-shock ν_m 比较。
F_{ν,max,r}reverse-shock peak flux densityJy 或 cgsshock crossing 附近可能大于 forward-shock peak flux。
ν_m^{IC}SSC characteristic frequencyHzThomson regime 下约为 2γ_m²ν_m。
ν_c^{IC}SSC cooling frequencyHz受 Klein-Nishina 和 cooling regime 影响。
F_{ν,max}^{IC}SSC peak flux densityJy 或 cgs量级上与 scattering optical depth 有关，完整谱形需数值计算。
τ_{γγ}photon-photon pair-production optical depthdimensionlessTeV photon escape 和 internal absorption 的关键量。
τ_{EBL}EBL optical depthdimensionlessTeV photon 在 extragalactic background light 中传播吸收的 optical depth。
R_STSedov-Taylor radiuscmblast wave 进入 non-relativistic transition 的量级半径。
t_NRnon-relativistic transition times, dlate radio calorimetry 常关心的时间尺度；通常是 gradual transition。
Πlinear polarization degreedimensionless 或 %synchrotron polarization diagnostic；需扣除 foreground / host contamination。
Π_maxmaximum synchrotron polarization degreedimensionless 或 %optically thin synchrotron 理论最大值，实际净 polarization 通常更低。
β_appapparent transverse velocity / cdimensionlessVLBI centroid motion 或 source expansion 的表观速度；可大于 1。
R_\perpsky-projected source sizecmVLBI angular size 转换得到的 projected size。
D_Aangular-diameter distancecm, Mpc, Gpcsource angular size 与 physical size 转换使用。
t_obsobserver-frame times, d通常含 redshift factor (1+z)；relativistic shell 常有 R / 2cΓ² 量级压缩。
Remission radius / shock radiuscm与 t_obs 的转换依赖 Γ、几何角度和动力学 regime。
zredshiftdimensionlessobserved frequency 除以 (1+z)，observed time 乘以 (1+z)。
D_Lluminosity distancecm, Mpc, GpcF = L / 4πD_L²；已包含 cosmological redshift effects。
Fenergy fluxerg cm^-2 s^-1可为 bolometric 或 band-limited，必须记录能段。
F_νspectral flux densityerg cm^-2 s^-1 Hz^-1 或 Jy与 L_ν 转换时有 (1+z) 因子。
Sfluenceerg cm^-2observer-frame time integrated flux；用于 E_iso 时需 band / K-correction。
L_isoisotropic-equivalent luminosityerg s^-1依赖 peak flux、time bin 和 bolometric correction。
Lluminosityerg s^-1photosphere radius 公式中通常指 isotropic-equivalent outflow luminosity。
Awind density normalizationg cm^-1wind medium 常写作 ρ(R)=AR^-2，也常用 A_* 归一化。
Δshell widthcmsource-frame shell 宽度，粗略对应 engine duration。
Tprompt duration / engine activity timescalesthin / thick shell 判据中常与 deceleration time 比较。
