# Konus-Wind

## 页面定位

本页用于组织 Konus-Wind 的仪器信息、数据产品和代表性结果。当前正式 ingest 的 source 覆盖 GRB 221009A 的 prompt emission、fluence、peak flux 和 isotropic-equivalent energetics。

## GRB 221009A 代表性结果

Frederiks et al. 报告 Konus-Wind 与 SRG/ART-XC 对 GRB 221009A 的观测和分析，并将该事件描述为 Konus-Wind 超过 28 年观测中探测到的最亮 gamma-ray burst。

已整理事实：

- Konus-Wind trigger time 为 2022-10-09 13:17:01.648 UTC；论文换算的 Earth-crossing reference time 为 13:17:00.401 UTC。
- pulsed prompt phase 持续约 600 s。
- prompt phase 后有 steady power-law decay，持续超过 25 ks。
- time-averaged spectral peak energy 为 Ep ≈ 2.6 MeV；最亮 emission peak 的 Ep ≈ 3.0 MeV。
- 20 keV–10 MeV 总 fluence ≈ 0.22 erg cm^-2；同能段 peak energy flux ≈ 0.031 erg cm^-2 s^-1。
- 采用 z = 0.151 时，论文给出 Eiso ≈ 1.2×10^55 erg，64 ms 尺度 Liso ≈ 3.4×10^54 erg s^-1。

## 使用 caveat

- Eiso 和 Liso 是 isotropic-equivalent 数值，依赖 redshift 和能段。
- “very hard, super-energetic version of a normal long GRB” 是作者解释，应与观测到的 fluence、flux、Ep 和持续时间分开。
