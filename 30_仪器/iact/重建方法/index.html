IACT 重建方法

IACT 重建方法

Section titled “IACT 重建方法”

当前范围

Section titled “当前范围”

本页记录 Hillas 1985 foundational source 中的单望远镜 image-parameter reconstruction / selection 思路，并用 de Naurois 2006 补充 Hillas scaled cuts、stereoscopic reconstruction、Model analysis 和 3D Model analysis。de Naurois & Rolland 2009 进一步给出 H.E.S.S. Model Analysis 的逐 pixel likelihood reconstruction、goodness-of-fit 定义和性能边界。Krawczynski et al. 2006 补充 VERITAS simulation context 下的 normalized width、fit-agreement、energy-consistency、timing 和 likelihood-ratio gamma/hadron separation。Bernlöhr et al. 2012 则把 reconstruction / selection 方法放到 CTA Production-1 array-scale simulations 中，用于比较 telescope classes、layout 和 sensitivity。

从 image axis 到 source discrimination

Section titled “从 image axis 到 source discrimination”

Hillas 1985 的 point-source 判据依赖一个几何事实：gamma-ray shower image 的主轴倾向于指向 camera 中的 source position；hadronic background 的 image orientation 则更分散。因此，image axis、source position 和 image centroid 的相对关系可用于方向一致性判别。

本 source 中直接使用的几何量包括：

  • LENGTH / WIDTH：image 的主轴与短轴 angular spread。
  • MISS：source position 到 image axis 的 perpendicular distance。
  • DISTANCE：image centroid 到 source position 的 distance。
  • AZWIDTH：沿 source-related 方位方向的 projected width。
  • FRAC(2)：最大两个 tube 的 signal concentration。

Early cut-based selection

Section titled “Early cut-based selection”

该 source 给出了一种早期 cut-based event selection：为 6 个 image parameters 各自定义 gamma-domain boundary，然后要求至少 4 个参数落入 gamma domain。作者报告该策略可拒绝约 60–70% proton background，同时保留大多数 gamma-ray showers。

这个选择逻辑说明 IACT 重建从一开始就不是单纯测量总光量，而是利用 image morphology、orientation 和 concentration 的组合。它也是后续多参数 cut、scaled-width / scaled-length、stereo geometry 和 machine-learning classifier 的历史前身。

Stereoscopic reconstruction

Section titled “Stereoscopic reconstruction”

de Naurois 2006 将 stereoscopic imaging technique 归因到 HEGRA 的工作：多个 telescope camera 中 shower image main axes 的交点给出 source direction，shower impact 也可用类似几何重建；energy 则由每个 telescope 的单望远镜能量估计加权平均得到。这一步把 Hillas image axis 从单镜 head-tail ambiguous 的几何量，升级为阵列级方向重建工具。

CTA array-scale simulation interface

Section titled “CTA array-scale simulation interface”

Bernlöhr et al. 2012 的 CTA design study 把 event reconstruction 放到阵列级 Monte Carlo workflow 中评估：CORSIKA / sim_telarray 产生 shower 和 telescope response，随后用 baseline Hillas / stereo analysis 及其他分析方法估计 candidate layout 的 effective area、angular resolution、energy resolution 和 point-source sensitivity。

该 source 强调 layout optimization 不能只看单一 event-level classifier。LST、MST、SST 在不同能段贡献不同：低能端由 LST 数量和 trigger threshold 主导，core energy range 由 MST array 的 event quantity / quality trade-off 主导，高能端由 SST 覆盖面积主导。因此 reconstruction method、telescope multiplicity、array geometry 和 background systematics 会共同决定 sensitivity。

Model analysis

Section titled “Model analysis”

Model analysis 用 semi-analytical shower development model 生成 image template，并逐 pixel 比较观测 image 与模型 image。de Naurois 2006 强调该方法使用所有 camera pixels，maximum likelihood 可同时重建 primary energy、direction 和 impact，而且不要求 image cleaning。其 gamma/hadron separation 使用由 likelihood 定义的 goodness-of-fit 变量 G。

VERITAS gamma / hadron separation parameters

Section titled “VERITAS gamma / hadron separation parameters”

Krawczynski et al. 2006 在 VERITAS simulation 中比较五类 gamma/hadron separation parameters。Normalized width 使用 Monte Carlo lookup table 中的 w_m(r,size) 和 w_90(r,size) 对每个 telescope 的 image width 归一化：

w=1Ntrig∑iNtrigwidthi−wm(ri,sizei)w90(ri,sizei).w = \frac{1}{N_{\rm trig}}\sum_i^{N_{\rm trig}}\frac{width_i-w_m(r_i,size_i)}{w_{90}(r_i,size_i)}.w=Ntrig​1​i∑Ntrig​​w90​(ri​,sizei​)widthi​−wm​(ri​,sizei​)​.

除 image width 外，该 source 还使用 direction / core reconstruction 的 fit agreement：χ_dir² 与 χ_core²；lateral Cherenkov light distribution 通过不同 telescope 的 energy estimates 是否一致来度量，可写作 energy-consistency χ_E²；timing information 则用 pixel signal rise-time spread 构造 χ_time²。这些参数之间相关性不强，因此可用 likelihood ratio 组合：

λ1=∑i=1N[ln⁡(Piγ)−ln⁡(Pip)],λ2=min⁡i[ln⁡(Piγ)−ln⁡(Pip)].\lambda_1 = \sum_{i=1}^{N}[\ln(P_i^\gamma)-\ln(P_i^{\rm p})], \qquad \lambda_2 = \min_i[\ln(P_i^\gamma)-\ln(P_i^{\rm p})].λ1​=i=1∑N​[ln(Piγ​)−ln(Pip​)],λ2​=imin​[ln(Piγ​)−ln(Pip​)].

该 source 报告三望远镜 trigger 下，最佳单参数 Q factor 约 1.7，五类方法组合后约 2.6；四望远镜 trigger 下，normalized width alone 约 Q=2.4，与 χ_core² 组合后约 Q=3.6。这些结果来自 simulated VERITAS data，应作为 method-performance comparison 而非观测事实。

Pixel likelihood reconstruction

Section titled “Pixel likelihood reconstruction”

de Naurois & Rolland 2009 将 Model Analysis 写成 H.E.S.S. context 下的 likelihood reconstruction。它将 shower model 给出的 expected signal μ 与每个 camera pixel 的 observed signal s 比较；pixel likelihood 把 photoelectron number 的 Poisson 分布与 PMT / pedestal Gaussian resolution 卷积：

P(s∣μ,σp,σγ)=∑nμne−μn!2π(σp2+nσγ2)exp⁡[−(s−n)22(σp2+nσγ2)].P(s|\mu,\sigma_p,\sigma_\gamma) = \sum_n \frac{\mu^n e^{-\mu}}{n!\sqrt{2\pi(\sigma_p^2+n\sigma_\gamma^2)}} \exp\left[-\frac{(s-n)^2}{2(\sigma_p^2+n\sigma_\gamma^2)}\right].P(s∣μ,σp​,σγ​)=n∑​n!2π(σp2​+nσγ2​)​μne−μ​exp[−2(σp2​+nσγ2​)(s−n)2​].

其中 σ_p 是 pedestal width，包含 electronics noise 和 night-sky background contribution；σ_γ 是 single-photoelectron peak width。该 source 强调每个 pixel 使用 measured σ_p 和 σ_γ，这是 Model Analysis 在 varying NSB 下较稳定的重要原因。

Fit 的输出包括 6 个 shower parameters：direction 两个参数、impact 两个参数、first interaction depth 和 energy；同时输出 correlation matrix / parameter uncertainties 和 final log-likelihood。用于 gamma / hadron discrimination 的 goodness-of-fit 写作：

G=∑i[ln⁡L(si∣μi)−⟨ln⁡L⟩∣μi]2 NdF.G = \frac{\sum_i[\ln L(s_i|\mu_i)-\langle\ln L\rangle|_{\mu_i}]}{\sqrt{2\,\mathrm{NdF}}}.G=2NdF​∑i​[lnL(si​∣μi​)−⟨lnL⟩∣μi​​]​.

NdF 在该 source 中定义为 pixel number minus 6；若 pixel likelihood 可视为独立随机变量，G 的期望为 0、方差为 1。实际 event selection 主要使用 ShowerGoodness 和 reconstructed primary interaction depth，并辅以 BackgroundGoodness、pure-NSB consistency 和 point-source θ² cut。

3D Model analysis

Section titled “3D Model analysis”

3D Model analysis 可视为 Hillas parameters 的三维推广：把 shower 建模为大气中的 Gaussian photosphere，并用 line-of-sight path integral 预测每个 pixel 的 collected light。该方法可重建 mean altitude、impact、direction、3D width / length 和 luminosity；其中 3D width / rescaled width 可用于 gamma/hadron separation。
