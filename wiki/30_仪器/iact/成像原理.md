# IACT 成像原理

## 核心图像

Hillas 1985 将 IACT 的基本判别问题表述为：TeV gamma-ray point source 诱发的 air shower 和 cosmic-ray nuclei / proton background 都会产生 Cherenkov light，但它们在 focusing mirror focal plane 中的 image morphology 不同，因此可以用图像的 width、length 和 orientation 等参数做背景抑制。

在该 source 的 Whipple Observatory 语境中，望远镜用 10 m reflector 和 photomultiplier camera 记录 shower image。模拟 gamma-ray image 的典型特征是 elongated image，其主轴大致指回 camera 中的 source position。

## Hillas-style image parameters

| 参数 | 含义 | 物理 / 分析用途 |
| --- | --- | --- |
| LENGTH | 沿 image major axis 的 weighted angular spread | 表征 shower image 的纵向延展。 |
| WIDTH | 垂直 major axis 的 weighted angular spread | gamma-ray shower 通常更窄，可用于区分 hadronic shower。 |
| MISS | source position 到 image axis 的垂直距离 | point-source gamma-ray image 应更接近指回 source。 |
| DISTANCE | image centroid 到 source position 的角距离 | 与 shower impact parameter、能量和 field-of-view 选择相关。 |
| AZWIDTH | 沿 source-related 方位方向投影后的 image width | 另一种把 orientation 与宽度合并的判别量。 |
| FRAC(2) | 两个最大信号 tube 占总信号的比例 | 表征 image concentration / compactness。 |

这些参数是后续 IACT image analysis 的基础语言。现代文献常把这一类二阶矩与方向参数统称为 Hillas parameters，但本页先保留该 1985 source 中的参数名和定义边界。

## Gamma / hadron separation 思路

Hillas 1985 的核心思路不是使用单一 cut，而是为多个 image parameters 定义 gamma-domain 边界，再要求事件满足其中若干条件。该 source 给出的示例是：6 个 image parameters 中至少 4 个落入 gamma domain，可拒绝约 60–70% proton background，同时保留大多数 gamma-ray showers。

这个结果应作为早期 foundational demonstration 使用，而不是现代 IACT 灵敏度或背景抑制性能指标。现代分析需要结合 stereoscopic geometry、instrument response、Monte Carlo templates 和更复杂的分类器。

## 图像与能量 / 视场

该 source 还指出，较高能量 shower 的 image centroid 可出现在离 source position 更远的位置，因此更高能量分析可能需要更大的 field of view。这个结论连接了 image morphology、camera geometry 和能量相关 acceptance。
