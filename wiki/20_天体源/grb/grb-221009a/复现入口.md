# GRB 221009A 复现入口

## 本页定位

本页连接 GRB 221009A 的数据文件、模型推导和代码复现。原则是先做可验证的简单模型，再逐步走向论文级模型。

代码根目录：`reproduce/grb/events/grb_221009a/`。

## 第一批 notebook / script 计划

1. `scripts/fit_lhaaso_broken_powerlaw.py`：复现 TeV light curve 的 onset / peak / break toy fit。
2. `scripts/check_forward_shock_scalings.py`：用 analytic scaling 生成数量级预测。
3. `notebooks/01_data_inventory.ipynb`：检查数据文件、单位和时间零点。
4. `notebooks/02_forward_shock_toy_model.ipynb`：从 textbook 公式到简单模型曲线。
5. `notebooks/03_model_comparison_notes.ipynb`：比较 LHAASO、Laskar、O'Connor 的参数定义。

## 人工验证记录

每个复现任务完成后，应在 `reproduce/grb/validation/` 建立验证记录，至少包含：对应理论页、对应代码、对应数据文件、单位检查、是否复现论文数量级、与论文图或参数的差异、用户人工检查意见。

## 当前限制

- 尚未抽取 model-ready light curve / SED 表。
- 目前代码只能作为模型骨架和 toy-level scaling，不应宣称已复现论文拟合。
- structured jet 完整复现通常需要角向积分和数值 afterglow code，本阶段只做参数定义和简单比较。
