# GRB 余辉符号表

| 符号 | 含义 | 常用单位 / 归一化 | 注意事项 |
| --- | --- | --- | --- |
| Γ | bulk Lorentz factor | dimensionless | 注意区分 initial Γ_0、shock Γ、component Γ。 |
| β | v/c | dimensionless | β = sqrt(1 - Γ^-2)。 |
| δ | Doppler factor | dimensionless | 依赖 viewing angle 和 Γ；`δ = [Γ(1-β cosθ)]^-1`。 |
| θ | emitting element 与观测者视线夹角 | rad | 小角近似下常与 `Γ^-1` 比较。 |
| θ_j | jet opening half-angle | rad 或 degree | small-angle approximation 下 f_b ≈ θ_j^2/2。 |
| E_iso | isotropic-equivalent energy | erg | 观测推断量，不等于 true energy。 |
| E | isotropic-equivalent or beaming-corrected kinetic energy | erg | 必须说明是动力学中使用的等效能量还是真实能量。 |
| E_0 | fireball 初始总能量 | erg | 用于定义 dimensionless entropy。 |
| M_0 | fireball 初始 baryonic mass | g 或 M_sun | baryon loading 的核心量。 |
| η | dimensionless entropy | dimensionless | `η = E_0 / (M_0 c²)`；baryonic fireball 中常给出 terminal Γ 的量级。 |
| Γ_0 | coasting shell 初始 Lorentz factor | dimensionless | afterglow onset / deceleration 公式中的初始 Lorentz factor。 |
| R_0 | central-engine 初始半径尺度 | cm | fireball acceleration scaling 的归一化半径。 |
| R_s | saturation radius | cm | `R_s ~ η R_0`，加速转入 coasting 的量级半径。 |
| R_ph | photosphere radius | cm | Thomson optical depth 降到约 1 的半径。 |
| R_dec | deceleration radius | cm | swept-up mass 足以显著减速 ejecta 的半径。 |
| n | ISM number density | cm^-3 | wind medium 中通常改用 A 或 A_*。 |
| n_ext | shock 前外部介质 number density | cm^-3 | shock jump condition 中的 upstream density。 |
| n' | shocked fluid comoving number density | cm^-3 | ultra-relativistic strong shock 中量级 `n' ~ 4Γ n_ext`。 |
| e' | shocked fluid comoving internal energy density | erg cm^-3 | afterglow synchrotron magnetic field 和 electron energy 的入口。 |
| k | external density power-law index | dimensionless | `ρ_ext = A R^-k`；ISM 为 0，wind 为 2。 |
| ρ_ext | circumburst medium mass density | g cm^-3 | 可写作 `n m_p` 或 `A R^-2`。 |
| m_sw | swept-up external mass | g | deceleration 和 BM energy scaling 的核心量。 |
| ε_e | electron energy fraction | dimensionless | shock internal energy 中进入 non-thermal electrons 的比例。 |
| ε_B | magnetic energy fraction | dimensionless | `B'^2/(8π)=ε_B e'`。 |
| B' | comoving magnetic field | G | shocked fluid frame 中的磁场。 |
| p | electron power-law index | dimensionless | `N(γ_e) ∝ γ_e^-p`；不要和 photon index 混淆。 |
| γ_e | electron Lorentz factor | dimensionless | comoving frame electron energy。 |
| γ_m | minimum injected electron Lorentz factor | dimensionless | 对应 synchrotron characteristic frequency `ν_m`。 |
| γ_c | cooling electron Lorentz factor | dimensionless | dynamical time 内刚好显著冷却的 electron Lorentz factor。 |
| ν_m | synchrotron injection / characteristic frequency | Hz | 由 `γ_m` 对应的 spectral break。 |
| ν_c | synchrotron cooling frequency | Hz | 由 `γ_c` 对应的 spectral break。 |
| ν_a | synchrotron self-absorption frequency | Hz | radio band 中常重要，本章只作为接口。 |
| F_ν,max | peak spectral flux density | Jy 或 erg cm^-2 s^-1 Hz^-1 | afterglow synchrotron spectrum 的归一化。 |
| Y | Compton parameter | dimensionless | inverse Compton cooling 对 `γ_c` 的修正。 |
| α | temporal decay index | dimensionless | 本教材采用 `F_ν ∝ t^-α ν^-β_spec`。 |
| β_spec | spectral index | dimensionless | 为避免和速度 `β=v/c` 混淆，谱指数写作 `β_spec`。 |
| Γ_ph | photon index | dimensionless | X-ray 文献中常有 `β_spec = Γ_ph - 1`，需检查定义。 |
| t_j | jet-break time | s, d | 常由 `Γ(t_j) ~ θ_j^-1` 估计。 |
| f_b | beaming fraction | dimensionless | 小角近似常用 `f_b ≈ θ_j²/2`，需检查 one-sided / two-sided convention。 |
| E_γ | beaming-corrected prompt gamma-ray energy | erg | 常由 `f_b E_{γ,iso}` 得到。 |
| E_k | afterglow kinetic energy | erg | 需说明是 isotropic-equivalent 还是 beaming-corrected。 |
| η_γ | prompt radiative efficiency | dimensionless | 常定义为 `E_γ/(E_γ+E_k)`。 |
| t_obs | observer-frame time | s, d | 通常含 redshift factor `(1+z)`；relativistic shell 常有 `R / 2cΓ²` 量级压缩。 |
| R | emission radius / shock radius | cm | 与 `t_obs` 的转换依赖 `Γ`、几何角度和动力学 regime。 |
| z | redshift | dimensionless | observed frequency 除以 `(1+z)`，observed time 乘以 `(1+z)`。 |
| D_L | luminosity distance | cm, Mpc, Gpc | `F = L / 4πD_L²`；已包含 cosmological redshift effects。 |
| F | energy flux | erg cm^-2 s^-1 | 可为 bolometric 或 band-limited，必须记录能段。 |
| F_ν | spectral flux density | erg cm^-2 s^-1 Hz^-1 或 Jy | 与 `L_ν` 转换时有 `(1+z)` 因子。 |
| S | fluence | erg cm^-2 | observer-frame time integrated flux；用于 `E_iso` 时需 band / K-correction。 |
| L_iso | isotropic-equivalent luminosity | erg s^-1 | 依赖 peak flux、time bin 和 bolometric correction。 |
| L | luminosity | erg s^-1 | photosphere radius 公式中通常指 isotropic-equivalent outflow luminosity。 |
| A | wind density normalization | g cm^-1 | wind medium 常写作 `ρ(R)=AR^-2`，也常用 `A_*` 归一化。 |
| Δ | shell width | cm | source-frame shell 宽度，粗略对应 engine duration。 |
| T | prompt duration / engine activity timescale | s | thin / thick shell 判据中常与 deceleration time 比较。 |
