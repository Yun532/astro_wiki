# GRB 080319B 瞬时辐射

## Prompt optical / gamma-ray 关系

Racusin et al. 2008 强调 GRB 080319B 的 prompt optical 和 gamma-ray light curves 在时间上相关，但 optical flux 远高于 gamma-ray spectrum 向低能的简单外推。这意味着 prompt optical emission 和 gamma-ray emission 不能被同一个简单谱成分直接连接。

## 多谱成分约束

该 source 将 optical / gamma-ray mismatch 解释为需要 distinct spectral components。对知识库使用而言，最重要的观测约束是：

1. prompt optical 与 gamma-ray emission 时间相关，说明两者可能与同一 central-engine episode 或相关 emission zones 有联系。
2. optical flux 过高，排除了把 gamma-ray spectrum 低能端简单延伸到 optical 的单成分解释。
3. 该 mismatch 支持多 emission components 或更复杂 radiation geometry，但不唯一指定具体辐射机制。

## 与双成分喷流的关系

Racusin et al. 将极端 apparent brightness 和 prompt / afterglow 行为放入 narrow core + wider jet picture 中解释。very narrow core viewed nearly on-axis 可解释极亮 prompt emission；但 prompt optical / gamma spectral mismatch 本身主要说明多 spectral components，而不是直接测量 jet angular structure。
