# 术语表

## 本页用途

本页用于维护中英文术语、推荐译名、缩写和使用边界。

## GRB jet structure 术语

| 术语 | 推荐写法 | 使用边界 |
| --- | --- | --- |
| top-hat jet / uniform jet | top-hat jet 或 uniform jet | 指 core 内能量和 Lorentz factor 近似均匀、边界外迅速下降的理想化喷流。 |
| structured jet | structured jet / 结构化喷流 | 指 dE/dΩ(θ,t)、Γ(θ,t) 等量随 polar angle θ 变化的喷流；不同 source 的参数化可能不同。 |
| two-component jet | two-component jet / 双成分喷流 | structured outflow 的离散化形式，通常含 narrow core 和 wider / slower component。 |
| jet core | jet core / 喷流 core | 角向结构中 Lorentz factor 和 energy density 近似均匀或最高的内区。 |
| jet wings | jet wings / 喷流 wings | core 外较宽区域，常含 mixed jet-cocoon material 或 cocoon-dominated material。 |
| viewing angle | viewing angle θ_v / θ_obs | 观测者视线与 jet axis 的夹角；必须和 opening angle、core angle 区分。 |
| quasi-universal structured jet | quasi-universal structured jet | population-level 假设：不同 GRB 的 angular structure 近似相似，观测差异部分来自 viewing angle。 |
