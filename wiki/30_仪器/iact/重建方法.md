# IACT 重建方法

## 当前范围

本页记录 Hillas 1985 foundational source 中的单望远镜 image-parameter reconstruction / selection 思路，并用 de Naurois 2006 补充 Hillas scaled cuts、stereoscopic reconstruction、Model analysis 和 3D Model analysis。de Naurois & Rolland 2009 进一步给出 H.E.S.S. Model Analysis 的逐 pixel likelihood reconstruction、goodness-of-fit 定义和性能边界。后续 C4–C5 source 会继续补充 VERITAS gamma/hadron separation 和 CTA simulation framework。

## 从 image axis 到 source discrimination

Hillas 1985 的 point-source 判据依赖一个几何事实：gamma-ray shower image 的主轴倾向于指向 camera 中的 source position；hadronic background 的 image orientation 则更分散。因此，image axis、source position 和 image centroid 的相对关系可用于方向一致性判别。

本 source 中直接使用的几何量包括：

- LENGTH / WIDTH：image 的主轴与短轴 angular spread。
- MISS：source position 到 image axis 的 perpendicular distance。
- DISTANCE：image centroid 到 source position 的 distance。
- AZWIDTH：沿 source-related 方位方向的 projected width。
- FRAC(2)：最大两个 tube 的 signal concentration。

## Early cut-based selection

该 source 给出了一种早期 cut-based event selection：为 6 个 image parameters 各自定义 gamma-domain boundary，然后要求至少 4 个参数落入 gamma domain。作者报告该策略可拒绝约 60–70% proton background，同时保留大多数 gamma-ray showers。

这个选择逻辑说明 IACT 重建从一开始就不是单纯测量总光量，而是利用 image morphology、orientation 和 concentration 的组合。它也是后续多参数 cut、scaled-width / scaled-length、stereo geometry 和 machine-learning classifier 的历史前身。

## Stereoscopic reconstruction

de Naurois 2006 将 stereoscopic imaging technique 归因到 HEGRA 的工作：多个 telescope camera 中 shower image main axes 的交点给出 source direction，shower impact 也可用类似几何重建；energy 则由每个 telescope 的单望远镜能量估计加权平均得到。这一步把 Hillas image axis 从单镜 head-tail ambiguous 的几何量，升级为阵列级方向重建工具。

## Model analysis

Model analysis 用 semi-analytical shower development model 生成 image template，并逐 pixel 比较观测 image 与模型 image。de Naurois 2006 强调该方法使用所有 camera pixels，maximum likelihood 可同时重建 primary energy、direction 和 impact，而且不要求 image cleaning。其 gamma/hadron separation 使用由 likelihood 定义的 goodness-of-fit 变量 G。

## Pixel likelihood reconstruction

de Naurois & Rolland 2009 将 Model Analysis 写成 H.E.S.S. context 下的 likelihood reconstruction。它将 shower model 给出的 expected signal `μ` 与每个 camera pixel 的 observed signal `s` 比较；pixel likelihood 把 photoelectron number 的 Poisson 分布与 PMT / pedestal Gaussian resolution 卷积：

```tex
P(s|\mu,\sigma_p,\sigma_\gamma) = \sum_n \frac{\mu^n e^{-\mu}}{n!\sqrt{2\pi(\sigma_p^2+n\sigma_\gamma^2)}} \exp\left[-\frac{(s-n)^2}{2(\sigma_p^2+n\sigma_\gamma^2)}\right].
```

其中 `σ_p` 是 pedestal width，包含 electronics noise 和 night-sky background contribution；`σ_γ` 是 single-photoelectron peak width。该 source 强调每个 pixel 使用 measured `σ_p` 和 `σ_γ`，这是 Model Analysis 在 varying NSB 下较稳定的重要原因。

Fit 的输出包括 6 个 shower parameters：direction 两个参数、impact 两个参数、first interaction depth 和 energy；同时输出 correlation matrix / parameter uncertainties 和 final log-likelihood。用于 gamma / hadron discrimination 的 goodness-of-fit 写作：

```tex
G = \frac{\sum_i[\ln L(s_i|\mu_i)-\langle\ln L\rangle|_{\mu_i}]}{\sqrt{2\,\mathrm{NdF}}}.
```

`NdF` 在该 source 中定义为 pixel number minus 6；若 pixel likelihood 可视为独立随机变量，`G` 的期望为 0、方差为 1。实际 event selection 主要使用 ShowerGoodness 和 reconstructed primary interaction depth，并辅以 BackgroundGoodness、pure-NSB consistency 和 point-source `θ²` cut。

## 3D Model analysis

3D Model analysis 可视为 Hillas parameters 的三维推广：把 shower 建模为大气中的 Gaussian photosphere，并用 line-of-sight path integral 预测每个 pixel 的 collected light。该方法可重建 mean altitude、impact、direction、3D width / length 和 luminosity；其中 3D width / rescaled width 可用于 gamma/hadron separation。

## 与现代重建的接口

| 早期 Hillas-style 量 | 现代分析中对应的问题 |
| --- | --- |
| WIDTH / LENGTH | shower image shape、gamma/hadron separation、scaled image parameters。 |
| image axis / MISS | arrival direction reconstruction、source consistency；stereo 中对应多 telescope image axes intersection。 |
| DISTANCE | impact parameter / camera offset / energy-dependent acceptance。 |
| FRAC(2) | image compactness、cleaning threshold 和 leakage / containment 问题。 |
| scaled cuts | Monte Carlo expectation based normalization；Mean Scaled Width / Length。 |
| model goodness G | pixel-level likelihood fit quality；用于 Model analysis gamma/hadron separation。 |
| ShowerGoodness / first interaction depth | de Naurois & Rolland 2009 中的 Model Analysis 主背景抑制量。 |
| 3D width / rescaled width | shower photosphere width；用于 3D Model analysis gamma/hadron separation。 |
| multiple parameter cuts | multivariate classifier 或 likelihood-based event selection。 |
